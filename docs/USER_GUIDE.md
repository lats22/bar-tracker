# Bar Tracker - User Guide

Welcome to Bar Tracker! This guide will help you get started with managing your bar's sales and expenses.

## Getting Started

### Installation on iPhone/iPad

1. Open Safari and go to your bar tracker URL
2. Tap the Share button (square with arrow)
3. Scroll down and tap "Add to Home Screen"
4. Name it "Bar Tracker" and tap "Add"
5. The app icon will appear on your home screen

Now you can open Bar Tracker like any other app!

### First Login

**Default Admin Credentials:**
- Username: `admin`
- Password: (set during installation)

**IMPORTANT:** Change the admin password immediately after first login!

## User Roles

### Staff
- View dashboard
- Add sales entries
- View sales and expenses history

### Manager
- All staff permissions
- Add/edit expenses
- View financial reports
- View activity logs

### Admin
- All manager permissions
- Create/manage user accounts
- Delete entries
- Full system access

## Dashboard

The dashboard shows you quick stats at a glance:

- **Today**: Sales, expenses, and profit for today
- **This Week**: 7-day summary
- **This Month**: Monthly summary

## Managing Sales

### Adding a Sale

1. Go to **Sales** page
2. Click **+ Add Sale**
3. Fill in:
   - **Date**: When the sale occurred
   - **Amount**: Total sale amount
   - **Payment Method**: Cash, Card, or Mobile
   - **Category**: Drinks, Food, or Other
   - **Notes**: Optional notes
4. Click **Save Sale**

### Viewing Sales

- Sales are listed in reverse chronological order (newest first)
- Filter by date range, category, or payment method
- Total sales shown at the top

### Payment Methods

- **Cash**: Physical currency
- **Card**: Credit/debit card payments
- **Mobile**: Digital wallets (Apple Pay, Google Pay, etc.)

### Sales Categories

- **Drinks**: Bar beverage sales
- **Food**: Food items
- **Other**: Merchandise, events, etc.

## Managing Expenses

**Note:** Only Managers and Admins can add/edit expenses.

### Adding an Expense

1. Go to **Expenses** page
2. Click **+ Add Expense**
3. Fill in:
   - **Date**: When the expense occurred
   - **Amount**: Expense amount
   - **Category**: Type of expense
   - **Vendor**: Supplier name (optional)
   - **Description**: What the expense was for
4. Click **Save Expense**

### Expense Categories

- **Supplies**: Bar supplies, ingredients, inventory
- **Utilities**: Electricity, water, internet
- **Salaries**: Staff wages
- **Rent**: Property rent/lease
- **Maintenance**: Repairs, equipment servicing
- **Other**: Miscellaneous expenses

### Viewing Expenses

- View all expenses for the selected period
- Total expenses shown at the top
- Filter by category or vendor

## Reports (Manager/Admin Only)

### Financial Reports

1. Go to **Reports** page
2. Select time period:
   - This Month
   - Last 30 Days
   - This Year

The report shows:

- **Total Sales**: All sales for the period
- **Total Expenses**: All expenses for the period
- **Net Profit**: Sales minus expenses
- **Profit Margin**: Profitability percentage

### Charts & Analytics

**Daily Trend Chart**
- Line chart showing sales, expenses, and profit over time
- Helps identify patterns and trends

**Sales by Category**
- Pie chart showing breakdown of sales by category
- See which products generate most revenue

**Expenses by Category**
- Bar chart showing expense distribution
- Identify biggest cost areas

## User Management (Admin Only)

### Creating New Users

1. Click on **Users** (admin only)
2. Click **+ Add User**
3. Fill in:
   - Username (unique)
   - Email
   - Password (minimum 6 characters)
   - Full Name
   - Role (Staff, Manager, or Admin)
4. Click **Create User**

### Managing Users

- View all users and their roles
- Edit user details (except username)
- Deactivate/activate user accounts
- Reset user passwords

## Best Practices

### Daily Tasks

**For Staff:**
1. Record all sales immediately after each transaction
2. Double-check amounts before submitting
3. Use correct payment methods and categories

**For Managers:**
1. Review daily sales totals
2. Record daily expenses
3. Check for any unusual activity

### Weekly Tasks

**For Managers:**
1. Review weekly reports
2. Compare with previous weeks
3. Verify all expenses are recorded
4. Check inventory vs. sales data

### Monthly Tasks

**For Admins/Managers:**
1. Generate monthly financial report
2. Analyze profit margins
3. Identify cost-saving opportunities
4. Review user activity logs
5. Backup database

## Security Tips

1. **Never share your password** with anyone
2. **Log out** when finished, especially on shared devices
3. **Use strong passwords** (mix of letters, numbers, symbols)
4. **Report suspicious activity** to admin immediately
5. **Change password regularly** (every 3 months recommended)

## Common Tasks

### Changing Your Password

1. Click on your name (top right)
2. Select "Change Password"
3. Enter current password
4. Enter new password (minimum 6 characters)
5. Confirm new password
6. Click "Change Password"

### Correcting a Mistake

**If you just entered wrong data:**
- Contact your manager/admin to edit or delete the entry
- Only managers and admins can edit/delete entries

**Prevention:**
- Double-check amounts before submitting
- Use notes field to add context

### Filtering Data

All lists (Sales, Expenses) can be filtered:
- By date range (start date and end date)
- By category
- By payment method (sales only)
- By vendor (expenses only)

## Offline Usage

Bar Tracker works partially offline:

- ✅ Can view previously loaded data
- ✅ App interface remains accessible
- ❌ Cannot add new entries without internet
- ❌ Cannot sync new data

**Note:** Make sure you have internet connection when entering data!

## Troubleshooting

### Can't log in
- Check username and password (case-sensitive)
- Contact admin if account is locked
- Clear browser cache and try again

### Data not appearing
- Check date filters
- Refresh the page
- Check internet connection
- Contact admin if problem persists

### App not loading
- Check internet connection
- Clear browser cache
- Try removing and re-adding to home screen
- Contact admin

### Charts not showing
- Ensure there is data for the selected period
- Try different date range
- Refresh the page

## Tips for Accuracy

### Sales Entry
- Enter sales in real-time when possible
- Keep receipt/order details handy
- Use notes for special transactions
- Group similar items in one entry

### Expense Entry
- Save receipts (take photos)
- Enter expenses same day when possible
- Be specific in descriptions
- Include vendor names

### Regular Reviews
- Check daily totals at end of shift
- Compare physical cash with system
- Report discrepancies immediately
